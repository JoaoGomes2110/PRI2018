Resumo:

Neste trabalho foi pedido para criarmos um Website. Para tal foi dado um ficheiro XML para analisarmos, este ficheiro continha o livro Quran.

Dividi a criação do Website em duas fases :

-> Geração do índice;

-> Geração das páginas individuais alcançadas a partir do índice.

Para a implementação da primeira fase criei uma tabela separando o índice, do conteudo. O índice foi numerado pela ordem das Suras, e o conteudo apresenta a informação relativa ao autor e ferramentas usadas, e ainda o prefácio.

A segunda fase é gerar as páginas individuais, isto é, gerar uma página para cada Sura. Desta forma, temos facilitada a tarefa de visualizar o conteúdo do livro, bem como, uma melhor organização do mesmo.
