Resumo:

Neste trabalho foi pedido que fosse implementada uma Aplicação Web para upload de ficheiros.

Para o desenvolvimento desta aplicação foi criado um servidor em Node Js para dar resposta aos pedidos do Cliente, para gerar as páginas web usamos templates em Pug.

Como desafio foi pedido que a aplicação web tivesse só uma página em vez de páginas separadas como estavamos habituados, segue em anexo o link git directamente para as duas versões criadas.

Resultados:

  1. Github de Tpc's de PRI para servidor com uma página Web
  2. Github de Tpc's de PRI para servidor com várias páginas Web
