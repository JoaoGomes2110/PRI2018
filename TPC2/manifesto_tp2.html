Resumo:

Neste trabalho apresentamos um DTD (Document Type Definition) sobre a página Web (http://clav.dglab.gov.pt/classes/consultar/c150.20.404) que traduz um processo de negócio. Este DTD vai permitir ajudar na validação, de processos que cheguem a página, evitando que processos que não cumpram as regras estabelicidades sejam rejeitados.

Fica então para ideia futura generalizar um pouco mais o DTD, de modo a, permitir validar vários tipos de processos diferentes cheguem ao Site.

Além disso é apresentado um DTD para o Manifesto deste trabalho que serviu para gerar e validar o ficheiro XML criado. Apresentamos ainda a Stylesheet criada para processar o ficheiro XML via XSLT.

O último ficheiro apresentado é uma página HTML, que é resultado da transformação feita via XSLT.
